# plate-orbcode

Visual canvas for OrbCode project maps in [NUU Steel](https://github.com/NUU-Cognition).

A Steel plate that renders codebase architecture as an interactive graph. Explore modules, dependencies, and semantic relationships on a DAG canvas. Pairs with the OrbCode shard.

## Install

```bash
flint plate install NUU-Cognition/plate-orbcode
```

## Features

- Interactive graph canvas (React Flow)
- Typed artifact nodes with semantic relationships
- Cross-plate action: update OrbCode from completed tasks
- Real-time updates via SSE
